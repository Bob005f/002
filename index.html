CheatSheet:

Klonen:

App: GitBash oder Powershell

GitLab Projekt öffnen --> Repository

Clon button --> SSH Key kopieren

Pfad eingeben beispiel; cd ./Downloads/

git clone + kopiertes eingeben und ausführen in git Bash

Beispiel: cd ./Projektname

Pushen:

git add * eingeben (* = alle Dateien oder nur : Dateiname)

git status (Grün = Es wurde geaddet, Rot = Es wurde nicht geaddet)

git commit -m &ldquo Kommentar &ldquo

git push

Tipp: cd.. = aus dem akutellen Ordner/verzeichniss einen schritt nach "hinten" Z.B. /Downloads/name/ zu /Downloads/


Branch

    git branch --> Zeigt mit * an auf welchem Branch man sich befindet

    git branch + Name --> Erstellt ein neues Branch

    git checkout + Name --> Wechselt aktuellen Branch zu dem eingegebenen Branch

    git push --set-upstream origin + Name --> Pusht neues Branch

    git merge + Name --> Führt (mergt) eingegebenes Branch zum aktuellen Branch zusammen, also man holt sich die Datein aus dem angegebenem Branch

    git branch -d + Name --> Löscht den angegebenen Branch
